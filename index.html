In the box

  • Open Source JavaScript framework (MIT)
  • Backend services agnostic
  • Windows, Linux, OSX
  • Comes with
    • StealJS - ES6, CJS, and AMD module loader and builder
    • CanJS - Custom elements and Model-View-ViewModel utilities
    • jQuery - DOM helpers
    • FuncUnit - Functional tests
    • Testee - QUnit, Mocha and Jasmine Test runner
    • DocumentJS - Documentation
    • can-ssr - Server-Side Rendering Utilities for CanJS
    • can-connect - Assemble real-time, high performance, restful data connections.
